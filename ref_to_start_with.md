## 6. Emerging Trends (2023–2024)
- **Self-supervised Learning**: Pre-training on unlabeled data (e.g., **PointContrast**).  
- **Transformers**: Hybrid architectures (e.g., **Point Transformer**).  
- **4D Segmentation**: Temporal modeling for dynamic scenes.

- **PyTorch Geometric** (for graph-based networks)  
- **Open3D** (for visualization and preprocessing)  
- **MMDetection3D** (for end-to-end pipelines)
